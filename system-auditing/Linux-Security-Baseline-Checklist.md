# ✅ Linux Security Baseline Checklist

Baseline items to review on Linux hosts.

---

## 🔐 Accounts & Auth

- remove unused accounts  
- enforce strong passwords  
- review sudoers

---

## 🛡️ Services & Network

- disable unused services  
- firewall default deny  
- monitor listening ports

---

## 📊 Logs & Monitoring

- auth/system logs  
- alerts for failed logins  
- centralized logging

---

## 📦 Updates

- automatic security updates  
- verify sources
